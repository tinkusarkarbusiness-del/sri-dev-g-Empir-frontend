# **App Name**: Sri Dev G Empire

## Core Features:

- Module Home: Provides a summary card and quick actions for each module, giving users a snapshot of key information.
- Detailed Module View: Presents detailed controls, live sensor data, and logs for each module, allowing for in-depth monitoring and management.
- Admin Dashboard: An Empire Control Room provides global KPIs, emotion heatmaps, and user management tools.
- UPI and Wallet Management UI: Cloud Function hooks that allow user control and visibility of their financial balance.
- Action Modal AI Assistant: Generate suggested actions (meditation, song, blessing) for a particular scenario using a generative AI tool, to enable Temple Mode, Mirror creation, and prediction execution.
- Secure User Authentication: Phone/email/Google Sign-up & Sign-in secured with MFA; onboard and manage user information with secure consent, suspend users and adjust user roles.
- Firebase Hosting: Allows connection of a custom domain in Firebase Hosting.

- Accent color: Divine Blue (#072A6D), is used on navigational elements and certain key pieces of content.
